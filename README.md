# vue-steam-chat

> A Vue.js component module

# Usage

    <vue-steam-chat text="Hello World!"></vue-steam-chat>

## Installation

### Using yarn

`yarn add vue-steam-chat`

### Using npm

`npm i --save vue-steam-chat`

## Demo and Docs

`npm run serve`
